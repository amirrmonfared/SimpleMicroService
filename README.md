# testMicroServices

## Overview
In this project, I gonna build some microservices to test them with integration tests.

### Setup infrastructure
run commands inside project directory.

- Start build containers:

    ```bash
    make up_build
    ```
    
  - Stop containers:

    ```bash
    make down
    ```

### How to run

- Run Front-end:

    ```bash
    make start
    ```
    
  - Stop Front-end:

    ```bash
    make stop
    ```
