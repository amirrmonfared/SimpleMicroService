# testMicroServices

## Overview
In this project, I gonna build some microservices to test them with integration tests.

### Setup infrastructure
run commands inside project directory.

### How to run microservices
- Start containers:

    ```bash
    make up_build
    ```
    
  - Stop containers:

    ```bash
    make down
    ```
  ### How to run tests
- Start test:

    ```bash
    make test
    ```

### How to run frontend service 

- Run Front-end:

    ```bash
    make start
    ```
    
  - Stop Front-end:

    ```bash
    make stop
    ```
